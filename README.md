## AI 使用方式

| 场景 | 使用方式 |
| --- | --- |
| 需求拆解 | 把模糊目标拆成可执行步骤，明确约束、风险和验证方式 |
| 编码实现 | 快速生成初稿，再结合上下文审查、改写和验证 |
| 调试定位 | 用日志、错误信息和最小复现缩小问题范围 |
| 文档与总结 | 把工程过程整理成清晰的说明、记录和复盘 |
| 自动化 | 用脚本和工具减少重复操作，让工作流更稳定 |
